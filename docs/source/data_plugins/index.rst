Data Plugins
============

Data plugins take data of certain format as input and produce structured data
that can serve as input for one of diagram plugins.

Data plugin device platform names correspond to
`Netmiko SSH device_type values <https://github.com/ktbyers/netmiko/blob/develop/PLATFORMS.md#supported-ssh-device_type-values>`_.
